_roles_and_permissions_validate_user()

_roles_and_permissions_validate_user(array  $form, array  $form_state)

User register form's validation additional callback function.

Validates if corresponding institution / ministry is selected for new user.

  • User with 'Valitsuse tasemel administraator' roll might not have any institution / ministry selected.
  • User with 'Ministeeriumi administraator' roll should have at list ministry to be selected.
  • User with 'Allasutuse administraator' should have both ministry and institution being selected.
